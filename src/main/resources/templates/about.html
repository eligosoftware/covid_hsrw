Data Source

The data about the numbers of the new confirmed cases, deaths, recpvered persons is get from the COVID-19 Statistics API. It is the Coronavirus COVID-19 Statistics API Based on public data by Johns Hopkins CSSE. It includes search by country, province, and date. Returns diff for confirmed, deaths, and recovered, compared with the previous day. To learn more click here.

The population of the countries was collected from Worldometers website for 2021. This information was used for the per 100.000 population chart

Used Libraries

The Javascript library JS Charting (free edition) was used for plotting.

Negative values

The number of negative values was detected during testing. They were not hidden and data was not modified. The user is notified when a negative value is detected.

Information Caching

The API renews data with some interval (unknown from the API documentation). To save time and not reach the same data every time, all the information from the API is cached for 15 minutes. After this period, new data is fetched from the API server.
